Components

NxMenuButtonComponent

Selector:

[nxMenuButton]

Properties
Name Description
@Input()

expandable: boolean

Whether this menu button is expandable or not. Will add a caret icon.
@Input()

expanded: BooleanInput

Whether this menu button is expanded or not. Only works in combination with the expandable option set to true.
@Input('nxType')

type: NxMenuButtonType

The type of this menu button. Can be primary or secondary, defaults to primary.

NxMenuComponent

This components is a collapsible menu for usage on smaller devices.

Can contain any content, but usually [nxMenuLink], [nxMenuGroup] and [nxAction].

Selector:

nx-menu

Properties
Name Description
@Input()

open: boolean

Whether the menu is open or closed.
