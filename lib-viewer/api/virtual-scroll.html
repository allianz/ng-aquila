setForOf( forOf: VirtualForOf )

Register the VirtualFor directive so viewport can call its methods. We register it instead of a contentChild query that it is available as early as possible and for edge cases when the directive would be wrapped in another component.

Imperatively update the spacer height on the DOM. This bypasses the async signal → computed → template binding cycle, ensuring the browser has the correct scrollHeight before scrolling.

Parameters:

height: number
