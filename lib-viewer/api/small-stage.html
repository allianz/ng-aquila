Properties
Name Description
@Input()

appearance: NxSmallStageAppearance

Expert option

Sets the appearance of the small stage.

Default: 'default'.
@Input()

narrow: BooleanInput

Reduces the width of the text to 6/12 instead of 8/12.

Only works with appearance = 'expert'..
