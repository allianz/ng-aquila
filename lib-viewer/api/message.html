Additional classes

NxMessageToastConfig

Configuration used when opening a message toast.

Properties
Name Description

announcementMessage?: string

Message to be announced by the LiveAnnouncer. When opening a toast message without a custom component or template, the announcement message will default to the specified message.

Default: ''.

context?: NxMessageToastContext

Context of the message toast.

Default: 'info'.

data?: D

Data being injected into the child component.

duration?: number

The length of time in milliseconds to wait before automatically dismissing the message toast.

Default: 3000.

politeness?: AriaLivePoliteness

The politeness level for the LiveAnnouncer announcement.

Default: 'polite'.
