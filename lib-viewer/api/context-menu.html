import { NxContextMenuModule } from '@aposin/ng-aquila/context-menu';

Directives

NxContextMenuContentDirective

Context menu content that will be rendered lazily once the menu is opened.

Selector:

ng-template[nxContextMenuContent]

NxContextMenuItemComponent

This directive is intended to be used inside an nx-context-menu tag. It exists mostly to set the role attribute, disabled state and styling.

Selector:

[nxContextMenuItem]

Exported as: nxContextMenuItem
Properties
Name Description
@Input()

disabled: boolean

Whether the context menu item is disabled.
Methods
focus(origin: FocusOrigin)

Focuses this context menu item.

Parameters:

origin?: FocusOrigin
getLabel(): string

Gets the label to be used when determining whether the option should be focused.

NxContextMenuTriggerDirective

This directive is intended to be used in conjunction with an nx-context-menu tag. It is responsible for toggling the display of the provided context menu instance.

Selector:

[nxContextMenuTriggerFor]

Exported as: nxContextMenuTrigger
Properties
Name Description
@Input('nxContextMenuTriggerFor')

contextMenu: NxContextMenuComponent

References the context menu instance that the trigger is associated with.
@Input('nxContextMenuTriggerData')

contextMenuData: Object

Data to be passed along to any lazily-rendered content.
@Input()

scrollStrategy: NxContextMenuScrollStrategy

@Output()

contextMenuClosed: EventEmitter

Event emitted when the associated context menu is closed.
@Output()

contextMenuOpened: EventEmitter

Event emitted when the associated context menu is opened.

contextMenuOpen: boolean

Whether the context menu is open.
Methods
closeContextMenu()

Closes the context menu.

focus()

Focuses the context menu trigger.

openContextMenu(origin: FocusOrigin)

Opens the context menu.

Parameters:

origin?: FocusOrigin
toggleContextMenu(origin: FocusOrigin)

Toggles the context menu between the open and closed states.

Parameters:

origin?: FocusOrigin
triggersSubmenu(): boolean

Whether the context menu triggers a sub-menu or a top-level one.

NxContextMenuComponent

Selector:

nx-context-menu

Exported as: nxContextMenu
Properties
Name Description
@Output()

closed: EventEmitter

Event emitted when the menu is closed.

parentMenu: NxContextMenuComponent | undefined

Parent menu of the current menu panel.
Methods
focusFirstItem(origin: FocusOrigin)

Focus the first item in the menu.

Parameters:

origin?: FocusOrigin
resetActiveItem()

Resets the active item in the menu. This is used when the menu is opened, allowing the user to start from the first option when pressing the down arrow.

Additional type aliases

NxContextMenuScrollStrategy

Type: 'close' | 'reposition'
