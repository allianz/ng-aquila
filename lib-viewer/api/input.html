NxPasswordToggleComponent

Selector:

nx-password-toggle

Properties
Name Description
@Input()

ariaLabel: string

Sets the aria-label needed for accessibility. Notice that this aria-label should describe the initial action according to the status of the visibility. E.g if you have an input[type=password] at the beginning then the password will be hidden and the correct aria-label would be 'Show password.'
@Input()

control: HTMLInputElement

Input element using the toggle functionality.
Methods
toggleInputType()

Toggles the type of the input.
