Services

NxConfigurationService

Service for switching values of the defaultOptions injection tokens at runtime. WARNING: This service should not be used in actual applications! It is only meant to showcase different options for documentation purposes.

Methods
clearOptions()
switchToExpert()
switchToRetail()

Additional interfaces

CloseScrollStrategyConfig

Config options for the CloseScrollStrategy.

Properties
Name Description

threshold?: number

Amount of pixels the user has to scroll before the overlay is closed.

NxScrollStrategyDefaultConfig

Scroll strategy default configuration.

Properties
Name Description

extraComponents?: { dialog?: Pick<NxScrollStrategyDefaultConfig, "scrollStrategy" | "scrollStrategyOptions">; overlay?: Pick<NxScrollStrategyDefaultConfig, "scrollStrategy" | "scrollStrategyOptions">; popover?: Pick<...>; tooltip?: Pick<...>; }

Configure default behavior for additional overlay-based component types.

overrides?: { autocomplete?: Pick<NxScrollStrategyDefaultConfig, "scrollStrategy" | "scrollStrategyOptions">; contextMenu?: Pick<NxScrollStrategyDefaultConfig, "scrollStrategy" | "scrollStrategyOptions">; datepicker?: Pick<...>; dropdown?: Pick<...>; notificationPanel?: Pick<...>; }

Override default config for each component type separately.

scrollStrategy: NxScrollStrategy

Preset for strategy factory that will be used to handle scrolling while a panel is open.

scrollStrategyOptions?: NxScrollStrategyOptions

Additional strategy preset options.

Additional type aliases

NxScrollStrategy

Scroll strategy default presets for overlay-based components.

Type: 'reposition' | 'block' | 'close' | 'noop'

NxScrollStrategyOptions

Scroll strategy default preset options for overlay-based components.

Type: RepositionScrollStrategyConfig & CloseScrollStrategyConfig

NxScrollStrategyFactoryFn

Scroll strategy factory creator function.

Type: (overlay: Overlay) => () => ScrollStrategy
