NxExpandableTableCellComponent

This is an expandable table cell. Only works in conjunction with the [nxExpandableTableRow] component.

NxExpandableTableRowComponent

This is an expandable table row. It can collapse its height to hide the content. Requires the table cells to have the [nxExpandableTableCell] component.

Directives

NxExpandableTableDirective

The nxExpandableTable adds row expansion functionality to the nxTable. It provides the logic to expand/collapse all expandable rows simultanously.

Selector: table[nxExpandableTable] ng-container[nxExpandableContainer]

Exported as: nxExpandableTable

Properties
Name Description

expanded: BehaviorSubject<boolean>

rows: QueryList<NxExpandableTableRowComponent>

Methods
close()

Closes all expandable rows.

expand()

Expands all expandable rows.

toggle()

Toggles the expandable rows. If all rows are open it will close them, otherwise it will open them.

NxSortDirective

Selector: table[nxSort]

Exported as: NxSortDirective

Properties
Name Description
Input @Input()

active: string

Sets the key of the most recently sorted column.
Input @Input()

direction: SortDirection

Sets the direction of the currently active sorted column. Default: 'asc'.
Output @Output()

sortChange: EventEmitter<SortEvent>

An event emitted when either the active sort or sort direction changes.
Methods
sort( sortable: string )

Sets the active sort key and determines the new sort direction. Afterwards the output event sortChange is called.

If sortable is not the key of the active sort header, the initial direction to sort is 'asc'. Otherwise the direction of the sorted column changes.

Parameters:

sortable: string
