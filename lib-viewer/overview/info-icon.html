The Info Icon component is a simplified wrapper around the Popover component, designed to display contextual information via an icon button. It requires Angular CDK (incl. CSS). Learn how to enable Angular CDK styles.

Scroll Strategy

The Info Icon component uses the Popover component internally. The scroll strategy is set to close by default, which means the popover will automatically close when the user scrolls. For further Information, refer to the Popover global settings documentation.
