Font weights

For the One Allianz design the font-weights of the headlines are already set as they should be. You should not need to change them anymore.

You can change the font-weight of any headline by using the font-weight utility classes nx-font-weight-{weight} as shown in the example below. Our library provides four font-weight options: light (300), regular (400), semibold (600) and bold (700).

Important: In order to use these utility classes you first have to import the utility css file in your project as explained here.

New explicit size input

Please note: In preparation for future design system updates for One Allianz the headline component gets new size names and a dedicated new size input with a proper typing. For existing NDBX projects you do not have to update to this API now, you can keep using the old one and we will keep it in the library for quite some time, even though it is marked as deprecated. Please also note that the t-shirt sizes are matching the One Allianz design and not the NDBX design. So if you want to use it already please double check if the font-size matches the design, as it will not necessarily match sizings in Figma for NDBX.

Example for the new API:

<!-- old usage -->
<h3 nxHeadline="section">
<!-- new usage -->
<h3 nxHeadline size="2xl">
