Watch it changing for tiny, small, medium, large, xlarge, 2xlarge and 3xlarge widths!

Use one input:

colOffset="2"
=> colOffset="2,2,2,2,2,2,2"

Use two inputs:

colOffset="2,3"
=> colOffset="2,3,3,3,3,3,3"

Use three inputs:

colOffset="0,3,4"
=> colOffset="0,3,4,4,4,4,4"

Use four inputs:

colOffset="0,3,4,5"
=> colOffset="0,3,4,5,5,5,5"
......

Use seven inputs:

colOffset="0,3,4,5,6,7,8"
=> colOffset="0,3,4,5,6,7,8"
