This example intends to show how to set explicit ariaLabels when you have multiple paginations, try it with a screenreader or look at the html code how the ariaLabel inputs are set. The example is not supposed to show how to have a working pagination on a table.

The following pagination gets the aria label set to "Cars table pagination"

Car Price
Porsche 72500

The following pagination gets the aria label set to "Customers table pagination"

Name ID
Max Mustermann 12345
