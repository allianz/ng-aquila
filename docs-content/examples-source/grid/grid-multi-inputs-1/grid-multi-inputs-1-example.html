Watch it changing for tiny, small, medium, large, xlarge, 2xlarge and 3xlarge widths!

Use two inputs:

nxCol="12,8"
=> nxCol="12,8,8,8,8,8,8"

Use three inputs:

nxCol="12,8,6"
=> nxCol="12,8,6,6,6,6,6"

Use four inputs:

nxCol="12,8,6,4"
=> nxCol="12,8,6,4,4,4,4"
......

Use seven inputs:

nxCol="12,8,6,4,3,4,3"
=> nxCol="12,8,6,4,3,4,3"
